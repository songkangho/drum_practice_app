# score_app
이 앱은 플러터와 파이썬을 이용한 벡엔드로 만들어 졌다.

## Project Structure
- **frontend/**: The Flutter application
- **backend/**: The API server
- **docs/**: Documentation, screenshots, and diagrams

# Features
- 악보의 오선보 인식
- 메트로놈 기능
- 메트로놈의 주기와 bpm에 맞는 현재 오선보 진행 상태 표시
- pdf 저장/불러오기
